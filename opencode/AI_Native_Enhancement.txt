10. KEY ARCHITECTURAL RECOMMENDATIONS
========================================================

1. Build Shared Services
Avoid duplicated logic across phases.

2. Introduce Event Bus Early
Critical for agent orchestration.

3. Keep Human Oversight
Mandatory approval for execution.

4. Prioritize Portfolio Intelligence
Optimize portfolio survival, not just entries.

5. Build Memory as Core Infrastructure
Long-term moat = accumulated intelligence.
