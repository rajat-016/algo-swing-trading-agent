---

## WHY THIS ORDER IS IMPORTANT

STEP 1 — REGIME AGENT

The Regime Agent provides:

* market context
* volatility awareness
* macro environment interpretation

Every downstream decision depends on this.

---

STEP 2 — SIGNAL AGENT

The Signal Agent validates:

* ML predictions
* historical similarity
* contextual trade quality

Requires:

* regime intelligence
* shared memory

---

STEP 3 — RISK AGENT

Risk validation must occur BEFORE:

* portfolio balancing
* execution workflows

This prevents unsafe portfolio states.

---

STEP 4 — PORTFOLIO AGENT

Portfolio Agent depends on:

* validated signals
* approved risk state
* exposure analysis

---

STEP 5 — GOVERNANCE FRAMEWORK

Human approval layer must sit AFTER:

* signal generation
* risk validation
* portfolio coordination

This ensures:

* execution safety
* institutional governance
* auditability

---

## KEY OUTPUTS

* autonomous market intelligence
* event-driven validation
* risk-aware workflows
* portfolio-aware reasoning
* human approval queues
* execution safeguards
